# Rreaing 11 - Controlling the size and aligning images with CSS.

## Controlling the size of an image with width and height. (img.small, medium, large)
Example: 
img.large {
  width: 500px;
  height: 500px;
}

## Aligning images with float. (left or right)
example:
img.align-left{
  float: left;
  margin-right: 10px;
}

## Centering an image.
example:
img.align-center{
  display: block;
  margin 0px auto;
}
## Background image
example:
body{
  background-image: url("image.gif");
}

## Repeat image (x-horizontally or y-vertically)
example:
body{
  background-image: url("image.gif");
  background-repeat: repeat-x;
}

## Positioning backgroung
useing left, right, center with top bottom center.
example:
body{
  background-image: url("image.gif");
  background-position: center top;
}
